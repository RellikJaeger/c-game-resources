# C Game Resources

The resources below were invaluable for the development of [Impaler](https://store.steampowered.com/app/1573090/Impaler/). I owe the success of the project to the talented open source developers who made their work available to the world.

#### Reference
 - [Quake 3 source code](https://github.com/id-Software/Quake-III-Arena)
 - [Awesome C](https://github.com/oz123/awesome-c)
 - [Awesome GLSL](https://github.com/vanrez-nez/awesome-glsl)

 - [Robby Kraft's C starter project](https://github.com/robbykraft/World)
